# Alarmio - App Description

## Short Description
Minimalist clock with ESP32 OLED display. Custom fonts, alarms, timer, and stopwatch via Bluetooth.

## Full Description

Transform your ESP32 into a beautiful desk clock with Alarmio. This app connects to your custom hardware via Bluetooth, turning a simple OLED display into a fully-featured timepiece.

**What Makes It Special:**
- Paint your own digit fonts on a 24×40 grid and send them to your clock
- Set multiple alarms with weekly repeat patterns
- Timer with animated progress ring that shows on your hardware
- Physical button control - dismiss alarms with a real button press
- Dark, minimalist interface that matches your hardware

**Features:**
✓ Bluetooth sync with ESP32
✓ Custom font painter
✓ Up to 5 alarms with repeat schedules
✓ Visual countdown timer with progress ring
✓ Precise stopwatch
✓ Automatic time & date sync
✓ Inline and painted font styles

**Perfect for:**
- Desk clock builds
- IoT projects
- Custom hardware interfaces
- Makers and tinkerers

Requires ESP32 with SH1107 128x128 OLED display (Arduino code included).

**Hardware Setup:**
Simple SPI connection - display, button on GPIO 9, and you're ready. Upload the Arduino code, pair via Bluetooth, and enjoy your custom clock.
